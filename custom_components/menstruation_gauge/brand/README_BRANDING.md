# Branding scaffold for `menstruation_gauge`

Lege hier deine finalen Bilddateien ab.

## Zielpfad in Home Assistant
`/config/custom_components/menstruation_gauge/brands/`

## Empfohlene Dateien
- `icon.png` (256x256)
- `icon@2x.png` (512x512)
- `logo.png` (256x256)
- `logo@2x.png` (512x512)
- `dark_icon.png` (256x256, optional)
- `dark_icon@2x.png` (512x512, optional)
- `dark_logo.png` (256x256, optional)
- `dark_logo@2x.png` (512x512, optional)

## Hinweise
- PNG mit transparentem Hintergrund verwenden.
- Quadratisches Format bevorzugen.
- Dateinamen exakt so schreiben.
- Nach dem Kopieren Home Assistant neu starten und Browser-Cache leeren.

## Minimal-Set
Wenn du nur 2 Dateien erstellen willst, nimm:
- `icon.png`
- `logo.png`
